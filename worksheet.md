2. How many times will the print statement be executed (in the asymptotic sense)
```
    for(int i = 0; i < n; i++) {
        for(int j = 0; j < n; j *= 2) {
            System.out.println("WHY IS THIS A QUESTION?");
        }
    }```

    1. O(logn)
    2. O(nlogn)
    3. O(n)
    4. O(n^2)
